# vue2-electron-quick-start

## Project setup
```
yarn install

npm install
```

### Compiles and hot-reloads for development
```
yarn electron:serve

npm run electron:serve
```

### Compiles and minifies for production
```
yarn electron:build

npm run electron:build
```
